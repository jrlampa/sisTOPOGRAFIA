# Template de Projeto - Industrial Orchestrator V2.0

Este diretório contém a estrutura padronizada para novos projetos de engenharia no sisRUA Unified.

## Estrutura de Fases

### 01-DADOS_BRUTOS
- Levantamentos de campo originais.
- Arquivos JSON/CSV brutos de APIs externas.
- Fotos e evidências locais.

### 02-PROCESSAMENTO
- Scripts de tratamento de dados.
- Cálculos intermediários de CQT e Tração.
- Simulações e cenários de engenharia.

### 03-ENTREGAVEIS
- Arquivos DXF finais editáveis.
- Memoriais descritivos.
- Listas de materiais (BOM).

### 04-RELATORIOS
- Relatórios de auditoria consolidados.
- PDFs de conformidade Light S.A.
- Análises de viabilidade AI.
