# Role: DevOps / QA (Stability & Testing)

## Responsibilities
- **Quality Assurance**: Maintaining Unit, E2E, and Chaos tests.
- **Infrastructure**: Managing Docker containers, `.dockerignore`, and deployment scripts.
- **Security**: Sanitizing data inputs and ensuring secure API communications.
- **DXF Validation**: Automated testing of generated DXF files using `accoreconsole.exe`.

## Key Directives
- Docker-first approach for all environment setups.
- Maintain >=80% test coverage for non-critical paths and 100% for critical engineering logic.
- Ensure `.gitignore` and `.dockerignore` are always up to date.
- Validate all public/free API integrations for "Zero Cost" compliance.
