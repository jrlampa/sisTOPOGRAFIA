# Role: Tech Lead (Orchestrator)

## Responsibilities
- **Orchestration**: Directing the integration between CQT, Tração, and Topografia modules.
- **Architecture**: Ensuring DDD principles and "Thin Frontend / Smart Backend" architecture are respected.
- **Standardization**: Enforcing the 4-phase project folder hierarchy (`orchestrator/template`).
- **SSoT**: Maintaining the "Single Source of Truth" via the Engineering Copilot Digital Manifest.

## Key Directives
- Always work on the `dev` branch.
- No mocked data in production or critical engineering paths.
- Enforce 2.5D standards (no 3D).
- Review all modularization for files exceeding 500 lines.
